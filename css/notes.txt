CSS:
    - Cascading style sheets
    - provides style to raw html
    - helps build responsive websites
    - Inline, internal and external ways of linking CSS
    - Inline is highest priority, then internal, then external

CSS Syntax:
    selector { property : value ;}
    example --> img { width: 30px;}

    - you can group p, a, and main tags together and apply the same style to them 
    p, a, main { background-color: white}

CSS selectors:
    - allows user / compiler to find HTML with properties to be set 

    Types of selectors:
        - element selectors
        - ID selectors
        - class selectors
        - grouping selectors
